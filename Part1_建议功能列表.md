# Part 1: 建议功能 (3%) - 导师查找系统

## 📋 功能规划概述

本系统将包含**8个核心功能模块**,涵盖用户管理、导师发现、预约管理、评价系统和激励机制。

---

## 🎯 详细功能列表

### 功能1: 用户注册与登录

| 项目 | 内容 |
|------|------|
| **功能名称** | 用户注册与登录 (User Registration & Login) |
| **描述/目的** | 允许学生创建账户并登录系统,确保只有在校学生可以访问平台 |
| **用户操作/功能** | • 使用学校邮箱注册账户<br>• 输入用户名、密码、专业、年级等信息<br>• 邮箱验证<br>• 使用邮箱和密码登录<br>• 记住登录状态 |
| **预期结果/显示** | • 注册成功后收到验证邮件<br>• 登录成功后跳转到主页/仪表板<br>• 显示欢迎消息和用户名<br>• 未验证邮箱会提示验证 |
| **备注/未来计划** | • Part 2: 前端表单验证<br>• Part 3: 连接数据库,实现JWT认证<br>• 未来: 支持第三方登录(Google, Microsoft) |

---

### 功能2: 个人资料管理

| 项目 | 内容 |
|------|------|
| **功能名称** | 个人资料管理 (Profile Management) |
| **描述/目的** | 允许用户创建和编辑个人资料,展示技能、专长和个人信息 |
| **用户操作/功能** | • 上传头像<br>• 填写个人简介<br>• 添加擅长的科目/课程(多个标签)<br>• 设置自己的角色(导师/学生/两者)<br>• 添加已完成的课程<br>• 编辑联系方式 |
| **预期结果/显示** | • 显示完整的个人资料页面<br>• 技能标签以标签云形式展示<br>• 显示作为导师的统计数据(辅导次数、评分等)<br>• 保存成功后显示确认消息 |
| **备注/未来计划** | • Part 2: 创建个人资料编辑表单<br>• Part 3: 保存到数据库,支持图片上传<br>• 未来: 添加简历/成绩单上传功能 |

---

### 功能3: 导师搜索与筛选

| 项目 | 内容 |
|------|------|
| **功能名称** | 导师搜索与筛选 (Tutor Search & Filter) |
| **描述/目的** | 帮助学生快速找到符合条件的导师,支持多维度搜索和筛选 |
| **用户操作/功能** | • 在搜索框输入科目/课程名称<br>• 使用筛选器:<br>  - 按科目分类<br>  - 按课程代码<br>  - 按评分排序(高到低)<br>  - 按可用时间筛选<br>• 查看搜索结果列表<br>• 点击导师卡片查看详情 |
| **预期结果/显示** | • 实时显示搜索结果<br>• 每个导师显示为卡片,包含:<br>  - 头像和姓名<br>  - 擅长科目标签<br>  - 平均评分(星级)<br>  - 辅导次数<br>  - "查看详情"按钮<br>• 无结果时显示"未找到导师"提示 |
| **备注/未来计划** | • Part 2: 实现搜索UI和筛选器组件<br>• Part 3: 连接后端API,实现实时搜索<br>• 未来: 智能推荐算法,基于学生需求匹配 |

---

### 功能4: 导师详情页

| 项目 | 内容 |
|------|------|
| **功能名称** | 导师详情页 (Tutor Profile Detail) |
| **描述/目的** | 展示导师的完整信息,帮助学生全面了解导师背景和能力 |
| **用户操作/功能** | • 查看导师完整资料:<br>  - 个人简介<br>  - 擅长科目列表<br>  - 已完成课程<br>  - 平均评分和评分分布<br>• 阅读其他学生的评价<br>• 查看导师可用时间日历<br>• 点击"预约辅导"按钮 |
| **预期结果/显示** | • 显示导师完整信息页面<br>• 评分以星级和数字显示<br>• 评价列表,每条包含评分、评论、日期<br>• 可用时间以日历形式展示<br>• "预约辅导"按钮突出显示 |
| **备注/未来计划** | • Part 2: 设计导师详情页面布局<br>• Part 3: 从数据库获取导师数据和评价<br>• 未来: 添加导师视频介绍功能 |

---

### 功能5: 预约辅导

| 项目 | 内容 |
|------|------|
| **功能名称** | 预约辅导 (Book Tutoring Session) |
| **描述/目的** | 允许学生预约导师的辅导时间,简化时间协调流程 |
| **用户操作/功能** | • 在导师详情页选择可用时间段<br>• 填写预约信息:<br>  - 选择科目/课程<br>  - 选择日期和时间<br>  - 描述需要帮助的内容<br>  - 选择地点(线上/线下)<br>• 提交预约请求<br>• 等待导师确认 |
| **预期结果/显示** | • 日历上可用时间段高亮显示<br>• 填写预约表单<br>• 提交成功后显示"预约已发送"确认<br>• 跳转到"我的预约"页面<br>• 发送邮件/通知给导师 |
| **备注/未来计划** | • Part 2: 创建预约表单和日历选择器<br>• Part 3: 保存到数据库,实现通知系统<br>• 未来: 集成Google Calendar, Zoom链接自动生成 |

---

### 功能6: 预约管理

| 项目 | 内容 |
|------|------|
| **功能名称** | 预约管理 (Booking Management) |
| **描述/目的** | 允许学生和导师查看、管理、取消或重新安排预约 |
| **用户操作/功能** | **学生端:**<br>• 查看我的预约列表(待确认/已确认/已完成/已取消)<br>• 取消预约<br>• 查看预约详情<br><br>**导师端:**<br>• 查看收到的预约请求<br>• 接受或拒绝预约<br>• 设置可用时间<br>• 管理时间表 |
| **预期结果/显示** | • 预约列表以卡片或表格形式展示<br>• 每个预约显示:<br>  - 对方姓名和头像<br>  - 科目和时间<br>  - 状态标签(待确认/已确认等)<br>  - 操作按钮(取消/查看详情)<br>• 不同状态用不同颜色标识<br>• 导师可一键接受/拒绝 |
| **备注/未来计划** | • Part 2: 创建预约管理界面<br>• Part 3: 实现状态更新和通知<br>• 未来: 添加预约提醒(24小时前/1小时前) |

---

### 功能7: 评价系统

| 项目 | 内容 |
|------|------|
| **功能名称** | 评价系统 (Rating & Review System) |
| **描述/目的** | 允许学生对完成的辅导进行评分和评论,建立信任机制 |
| **用户操作/功能** | • 辅导完成后收到评价提醒<br>• 给导师打分(1-5星)<br>• 撰写文字评论<br>• 提交评价<br>• 查看自己收到的评价(导师视角)<br>• 查看自己给出的评价(学生视角) |
| **预期结果/显示** | • 辅导完成后弹出评价表单<br>• 星级评分组件(可点击选择)<br>• 文本输入框撰写评论<br>• 提交后显示"感谢您的反馈"<br>• 评价立即显示在导师资料页<br>• 导师的平均评分自动更新 |
| **备注/未来计划** | • Part 2: 设计评分组件和评论表单<br>• Part 3: 保存评价到数据库,计算平均分<br>• 未来: 添加举报不当评论功能,防止刷分机制 |

---

### 功能8: 积分与徽章系统

| 项目 | 内容 |
|------|------|
| **功能名称** | 积分与徽章系统 (Points & Badges System) |
| **描述/目的** | 通过游戏化激励导师参与,增加平台活跃度和用户粘性 |
| **用户操作/功能** | • 完成辅导自动获得积分<br>• 获得高评分获得额外积分<br>• 达到里程碑解锁徽章:<br>  - "新手导师" (完成1次辅导)<br>  - "优秀导师" (平均评分4.5+)<br>  - "资深导师" (完成10次辅导)<br>  - "学霸" (辅导5门不同科目)<br>• 查看积分排行榜<br>• 在个人资料展示徽章 |
| **预期结果/显示** | • 个人资料页显示总积分和徽章<br>• 完成辅导后弹出"恭喜获得XX积分"<br>• 解锁徽章时显示动画和祝贺消息<br>• 排行榜页面显示前20名导师<br>• 徽章以图标形式展示在导师卡片上 |
| **备注/未来计划** | • Part 2: 设计积分和徽章展示UI<br>• Part 3: 实现积分计算逻辑和徽章解锁系统<br>• 未来(高级功能): 积分可兑换奖励,每月导师之星评选 |

## 🎨 用户流程图

```
新用户注册
    ↓
邮箱验证
    ↓
完善个人资料 (添加技能/科目)
    ↓
选择角色: 寻求帮助 / 提供帮助
    ↓
┌─────────────────┬─────────────────┐
│   寻求帮助      │   提供帮助      │
│   (学生)        │   (导师)        │
└─────────────────┴─────────────────┘
         ↓                   ↓
    搜索导师            设置可用时间
         ↓                   ↓
    查看详情            等待预约请求
         ↓                   ↓
    预约辅导      ←─────  接受/拒绝
         ↓                   ↓
    完成辅导        ←─────→  完成辅导
         ↓                   ↓
    评价导师                获得积分
         ↓                   ↓
   查看历史                解锁徽章
```
